Growth Chart Concepts

Five combo (bar + line) designs comparing post-based growth across Metal, Wood, Water, Fire and Earth. Pick a winner, then we wire it into the dashboard.

Loading data…

01 Grouped Bars (Volume) + 5 Growth-% Lines

Five colored bars per week (one per element, left axis) show that week's post volume. Five matching lines (right axis, 0–100%) trace each element's own cumulative growth — indexed so each line reaches 100% at its own all-time total — so you can see who posted the most this week and whose growth is front-loaded vs. still accelerating.

02 Stacked Bars + Per-Element Growth Lines

Stacked bars show each week's post composition across all five elements. Five thin overlaid lines trace each element's own cumulative growth, sharing a secondary right-hand scale. Best for seeing which element is pulling ahead over time.

03 Small-Multiples Element Cards

One mini combo chart per element (weekly bars + cumulative sparkline), styled like the existing dashboard cards. Best fit for dropping straight into the current element-grid layout without a big new section.

04 Ranking Bars + Growth-Rate Line (Dual Axis)

Elements ranked left-to-right by all-time post count (bars, left axis). A second line plots each element's growth rate — recent weeks vs. earlier weeks — around a zero baseline (right axis). Best for "what's winning, and what's accelerating" at a glance.

05 Hero-Style Glow Timeline

Marketing-style full-width chart matching the landing page's dark hero aesthetic: soft background bars for total weekly volume, five glowing smooth lines for each element's cumulative growth, with end-of-line value markers. Best for the homepage hero, not a dense analytics view.
